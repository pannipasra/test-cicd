## Test for CI/CD

### How to run
first have to enter `./app`
```sh
cd ./app
```

in typical mode
```sh
fastapi run main.py
```
in dev mode
```sh
fastapi dev main.py
```

### Testing
simply run **pytest**
```sh
pytest
```
